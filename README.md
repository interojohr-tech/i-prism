# 인사평가 시스템 v2 (수정 버전)

## 수정 사항

### 1. UI 가독성 및 사용자 편의성 개선
- 탭 레이블 명확화 (기능 목적에 맞게 재명명)
- 페이지별 상세 설명 추가
- 종합평가 테이블 순위·점수 시각화 강화

### 2. 본부장 — 종합평가 기능 (기존 "평가 결과 제출하기" 대체)
- **잠금 해제 조건**: 하위 구성원 전원의 자기평가 + 1차 + 2차 평가 완료 시 활성화
- 미완료 항목 목록을 명시적으로 표시
- 팀원 점수 **내림차순 정렬** 표시
- 등급(S/A/B/C/D) **직접 배분** 가능
- **자동 등급 배분** 버튼: 배분율 가이드(A 20%, B 50%, C 20%, D 10%) 기준 자동 계산
- 상단 등급 배분 현황 바 + 가이드 vs 현재 비교 테이블 실시간 반영
- 확정 제출 시 사장/회장 승인 단계로 전달

### 3. 사장·회장 — 최종 승인 기능 (신규 탭)
- **탭명**: 최종 승인
- **필터**: 부서별 / 등급별 필터링
- **통계 시각화**:
  - 등급별 인원 요약 카드 (5개 등급)
  - 전체 등급 분포 바
  - 부서별 평균 점수 + 진행 바
  - 부서별 등급 분포 테이블
- **구성원 상세 목록**: 종합점수 내림차순, 자동등급/최종등급 표시
- **승인 순서**: 종합평가 제출 → 사장 승인 → 회장 최종 확정
- 회장 최종 확정 시 전체 평가 결과 자동 공개
